# ProyectoFinal_TuBalance

## Introducción
**TuBalance** es una página web financiera diseñada para ayudar a los usuarios a controlar sus ingresos y gastos, visualizar el equilibrio entre activos y pasivos, y recibir consejos personalizados de ahorro.  
Su propósito es fomentar hábitos financieros saludables y brindar herramientas claras para alcanzar metas económicas de manera organizada y confiable.

## Descripción
Landing page desarrollada como parte del portafolio académico en GitHub.  
Incluye estructura en HTML, estilos en CSS y funcionalidades básicas en JavaScript.  
El proyecto refleja un flujo de trabajo real con commits narrativos, Issues y despliegue en GitHub Pages.

## Metodología de desarrollo
El proyecto se trabajó en 5 fases:
1. **Planeación**: definición de estructura y secciones.  
2. **Diseño**: aplicación de paleta de colores y tipografía.  
3. **Implementación**: desarrollo de HTML, CSS y JS.  
4. **Gestión**: creación de Issues, commits narrativos y tablero Kanban.  
5. **Entrega**: despliegue en GitHub Pages y documentación final.
